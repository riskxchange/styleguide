Forms

Form Basics

Forms can be created using .rx-form. They can have a title .rx-form__title and actions .rx-form__actions which can contains submit buttons, etc.

Forms contain fields .rx-field, which can contain inputs .rx-input, selects .rx-select, radios .rx-radio-group, checkboxes .rx-checkbox-group and their corresponding labels .rx-label.

Fields can also include info about the field, using .rx-field__info.

Tooltip

All tooltips .rx-tooltip should be used within .rx-field elements.

On mobile, they require the icon to be clicked in order to toggle the .rx-tooltip--active selector, which shows/hides the tooltip.
